## 🎯 프로젝트 개요

> **크래비스, 바슬러, 콤퓨타, LVS 등 글로벌 브랜드 머신비전 솔루션을 제공하는 전문기업 홈페이지**

허밍비전은 산업용 머신비전 분야의 혁신적인 시각 솔루션을 제공하는 전문기업입니다.  
본 프로젝트는 **요구사항 분석부터 인프라 설계까지 전 과정을 직접 설계**하여,  
고객과 관리자 모두가 편리하게 사용할 수 있는 **완전한 웹 플랫폼**을 구축했습니다.
